# PRMN2
a full rewrite of the original [prmn](https://github.com/chickencuber/project-manager)  
only works on unix-like systems 
## Features
- create projects with a template
- use categories to organize projects
- delete/rename projects
- search for a project using a built in fuzzy finder
- full mouse support
## Differences between this and the original
- the original codebase was held together with duct tape and hope
- the original lacked error handling
- this added renaming projects
- this added mouse support
